About

This is a minimal, one-folder control UI for the UNO MAS swarm (soil sampling robots). It runs a local simulator by default, and exposes a clean API for later ROS2 integration.

  • Set a rectangular field and sampling grid
  • Register robots by serial
  • Start, Pause, Resume, Return-to-Base
  • Live updates via Server-Sent Events
